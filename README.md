# VTBenchLab

VTBenchLab is a benchmarking lab for *visual tokenizers*.

This project collects and organizes multiple visual-tokenizer-related benchmarks across different evaluation categories, and provides a unified pipeline for setting up, adapting, and running these benchmarks in new environments.

Since visual tokenizers in our project may come from different sources and follow different interfaces, VTBenchLab also provides several example visual tokenizer adapters. These examples are intended to help users quickly plug in their own visual tokenizer and evaluate it under a consistent benchmark pipeline.
